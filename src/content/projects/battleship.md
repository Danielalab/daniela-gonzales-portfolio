## About Code Tips

A fully interactive, browser-based version of the classic strategy game Battleship. This project features dynamic board rendering, hit-detection logic, and a turn-management system. Players must strategically target coordinates to sink the hidden fleet before exhausting their limited ammunition.

## Core Features

- Core Functionality: The application allows the user to play the famous turn-based naval combat game entirely within a web browser.
- Gameplay Mechanics: The user interacts with a game board to take shots, attempting to locate and sink all of the opponent's ships.
- Win/Loss Conditions: The game concludes when the user successfully sinks all opposing ships or runs out of their allocated number of turns/moves.
